Docker Toolbox

Docker Toolbox is an installer for various Docker components on Apple Mac OS and Microsoft Windows.

See GitHub for more details.

Features

Users can pick and choose from the following to install from the Toolbox: (default: all)

  • Docker Client
  • Docker Machine
  • Docker Compose (Mac only)
  • Docker Kitematic
  • Oracle VirtualBox 5
